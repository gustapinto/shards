# Shards

<img src="docs/images/cover.png">

An client-side adhoc database/sql sharding library

## Installation

```bash
go get github.com/gustapinto/shards@v0.2.1
```

### API status

Currently shards API is not set on stone yet, so we highly recommend you to use explicit versions of shards when using this library (ex: `go get github.com/gustapinto/shards@X.Y.Z` instead of just `go get github.com/gustapinto/shards`).

The available versions can be found [here](https://github.com/gustapinto/shards/tags)
